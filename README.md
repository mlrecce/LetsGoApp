# LetsGoApp

A React Native travel app — your information source during a vacation.

## Getting Started

### Prerequisites

- Node.js >= 18
- React Native CLI or Expo CLI
- iOS: Xcode 15+
- Android: Android Studio with SDK 34+

### Installation

```bash
git clone git@github.com:mlrecce/LetsGoApp.git
cd LetsGoApp
npm install
```

### Running the App

```bash
# iOS
npx react-native run-ios

# Android
npx react-native run-android
```

## Project Structure

```
src/
├── screens/        # App screens
├── components/     # Reusable UI components
├── navigation/     # React Navigation config
├── services/       # API clients & external services
├── utils/          # Helper functions
└── assets/         # Images, fonts, etc.
```
